# Cute Ghost Cup / Desk Organizer 🐙👻

3D printable cute ghost-shaped cup or desk organizer. Holds a standard 12oz can, pens, or small items. Designed for easy printing.

## Features
- Hollow interior for functionality
- Simple debossed face
- Wavy ghost base
- Fully customizable OpenSCAD source
- Manifold and print-ready

## Files
- `ghost_cup.scad` - Main OpenSCAD model
- `renders/` - Preview images
- `stls/` - Exported STL files (coming soon)

## Printing Instructions
- Material: PLA/PETG
- Layer height: 0.2mm
- Infill: 15-20%
- Supports: Minimal

## Customization
Open in OpenSCAD and tweak parameters like height, diameter, etc.
